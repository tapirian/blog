# 使用U盘安装windows操作系统

## 啥是操作系统
 操作系统是是管理计算机硬件与软件资源的计算机程序。

## Windows安装 
1. 首先我们需要到网上下载你想要安装的操作系统的镜像文件：
推荐网站：[http://msdn.itellyou.cn/?lang=zh-cn](http://msdn.itellyou.cn/?lang=zh-cn)
2. 其次我们需要将镜像文件写入U盘，将U盘制作为启动盘：
推荐软件(软碟通)：[https://pan.baidu.com/s/1kcAOoZlHGcFN3PeIsfrBIw ](https://pan.baidu.com/s/1kcAOoZlHGcFN3PeIsfrBIw )
*提取码：7rlg*
3. 插入系统盘，开启电脑，进入bios界面。（不同品牌型号的电脑启动方式不同，自行百度。我的是dell，启动电脑后不停点击F2进入）
4. 在启动项中设置中，将USB启动调至最前方。F10保存退出。
5. 根据界面提示自行操作。
