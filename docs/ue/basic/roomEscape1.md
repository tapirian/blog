# 实现一个解谜小游戏（第一关）

## 效果说明
角色想要从复活点进入房间，需要找到房间钥匙。找到钥匙进入房间，然后把壁灯全部打开，后门打开。然后坐电梯上二楼（按数字键`2`键上二楼，按数字键`1`键回一楼）。
