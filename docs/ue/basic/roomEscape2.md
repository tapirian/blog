# 实现一个解谜小游戏（第二关）

## 效果说明
角色需要先踩到岩石地板上边，触发地板旁边的射灯，射灯照在墙上，同时显现墙上的按钮开关。角色需要到餐桌上吃掉汉堡，角色会变高变大，然后再去墙上鼠标点击开关，开启魔界之门，进入下一关。
